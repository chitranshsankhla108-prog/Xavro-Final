# CLAUDE.md

You are working on the XAVRO website.

You must act as a senior creative frontend engineer, motion designer, UI/UX designer, and visual QA reviewer.

This is not a normal landing page build.
This is a premium dark editorial single-page scroll experience for XAVRO, a done-for-you Instagram content agency for fitness coaches.

The most important rule:
Do not only code. After every meaningful task, you must screenshot the result, visually review it, analyze what worked and what failed, then fix the issues before moving forward.

## 1. Project Direction

XAVRO should feel like:

* premium
* cinematic
* dark editorial
* luxury but sharp
* athletic but not loud
* Swiss-influenced but not poster-cloned
* high-performance
* smooth
* confident
* one continuous brand film

XAVRO should not feel like:

* a generic agency landing page
* a SaaS website
* a presentation
* a slide deck
* a dashboard
* a glassmorphism UI
* a cyberpunk site
* a template
* disconnected sections

Core brand promise:

You coach. We handle the content.
